# IoT Lab 4 - Aplicación de Recetas

Este proyecto es una aplicación Android desarrollada para el **Laboratorio 4** del curso de IoT. La aplicación permite explorar categorías de comida, buscar platos por ingredientes o ID, y obtener recetas aleatorias mediante el uso de sensores.
